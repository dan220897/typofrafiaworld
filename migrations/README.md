# Миграции прайс-листа типографии

## Описание

Этот набор SQL миграций добавляет полный прайс-лист типографии в базу данных с учетом себестоимости, маржи и рентабельности.

## Структура миграций

### 002_alter_services_structure.sql
**ВАЖНО!** Изменяет структуру таблицы services:
- Меняет тип `id` с INT на VARCHAR(50)
- Добавляет поле `label` (название услуги)
- Добавляет поле `icon` (иконка)
- Очищает старые данные из таблицы services

### 003_add_cost_prices.sql
Добавляет поля для учета себестоимости во все таблицы параметров.

### 004_full_pricelist_part1.sql
**Категории:** Печать документов, Копирование, Сканирование

### 005_full_pricelist_part2.sql
**Категории:** Визитки, Листовки, Брошюры

### 006_full_pricelist_part3.sql
**Категории:** Плакаты, Наклейки, Широкоформатная печать

### 007_full_pricelist_part4.sql
**Категории:** Чертежи, Фотоуслуги, Календари, Дипломы, Открытки, Дизайн

## Как применить

```bash
mysql -u USER -p anikannx_printtg < migrations/full_pricelist_all.sql
```
